5. Lightshot features


Here is the list of features of Lightshot so you know all the things you can do with it:

  • Pencil (1) : It allows you to draw on the screenshot while you hold down the click.
  • Line (2) : It allows you to draw a straight line on the screenshot.
  • Arrow (3) : It allows you to draw an arrow on the screenshot.
  • Rectangle (4) : It allows you to draw a rectangle on the screenshot.
  • Highlight (5) : It allows you to highlight things on the screenshot.
  • Text (6) : It allows you to write text on the screenshot.
  • Color (7) : It allows you to select the color of the things you draw on the screen shot.
  • Undo (8) : It undo the most recent draw on the screenshot.
  • Save (9) : It allows you to save a screenshot into your local.
  • Copy to clipboard (10) : It allows you to save a screenshot on the clipboard.
  • Print (11) : It allows you to print the screenshot.
  • Search in google (12) : It allows you to find similar images using google.
  • Share (13) : It allows you to share a screenshot on your social networks.
  • Upload (14) : It allows you to upload your screenshot to the cloud.
  • Exit (15) : It allows you to exit from Lightshot.
